HTML Basics

HTML (HyperText Markup Language) is the foundation of web development. It provides the structure and content of web pages.

Key Concepts:
  • HTML elements and tags
  • Document structure (html, head, body)
  • Common elements: headings, paragraphs, links, images
  • Semantic HTML5 elements
Example:
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Welcome</h1>
    <p>This is a paragraph.</p>
</body>
</html>
{% elif skill.name == "CSS Basics & Layouts" %}
CSS Fundamentals

CSS (Cascading Style Sheets) is used to style and layout HTML elements. It controls colors, fonts, spacing, and positioning.

Key Concepts:
  • Selectors and properties
  • Box model (margin, padding, border)
  • Display and positioning
  • Flexbox and Grid layouts
Example:
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    padding: 20px;
}

.button {
    background-color: #007bff;
    color: white;
    padding: 10px 20px;
    border-radius: 5px;
}
{% elif skill.name == "JavaScript Fundamentals" %}
JavaScript Basics

JavaScript is a programming language that adds interactivity to web pages. It can manipulate HTML elements and respond to user actions.

Key Concepts:
  • Variables and data types
  • Functions and scope
  • DOM manipulation
  • Event handling
Example:
// Select an element
const button = document.querySelector('button');

// Add event listener
button.addEventListener('click', function() {
    alert('Button clicked!');
});

// Change content
document.getElementById('title').textContent = 'New Title';
